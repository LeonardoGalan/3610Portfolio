Technical Project

GameRator - Video Game Rating Website

GameRator
  • Developed a website in a team of 4 that allows users that have signed up through Google to rate currently trending video games from up to 5 different genres.
  • Used the RAWG video game database API to retrieve information of popular games, as well as the Google Cloud API to implement a Google user login for the frontend.
  • Set up the Express server and API routes for communication between the frontend and backend.
  • Created the backend database models/logic with Sequelize which helped seed data from the RAWG API.
